# Hravý přírodopis 7

A simple, interactive static website for learning 7th-grade natural science (biology).

## Project Overview
- **Goal:** Provide an engaging way for students to learn about zoology (vertebrates) and botany.
- **Features:**
  - Interactive topic cards with descriptions and examples.
  - Built-in quiz system for knowledge verification.
  - Responsive design using vanilla CSS.
- **Tech Stack:** HTML5, CSS3, and Vanilla JavaScript.

## File Structure
- `app.html`: The main interactive application with full features and quizzes.
- `index.html`: A simplified version or alternative landing page.
- `serve.sh`: A helper script for local development.
